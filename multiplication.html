Multiplication

What is multiplication?

Multiplication of two numbers is adding the first number to the second number, the second number of times. It is also called as repeated addition.

It is denoted by the ‘x’ symbol.

To perform Multiplication, one needs to know Addition properly, which can be learnt here.

Now,Let’s take an example:

Q] What is the answer of 4 x 3?

Answer: Here, the first number is 4 And second number is 3

So, the question can be read as ‘4 cross 3’ or ‘4 multiplied by 3’

Now, to solve this let’s take an example:

How many total balls are there, if we have 4 bags full of 3 balls each?

There are 4 bags, each having 3 balls, and we need to find total number of balls

Let us visualize this expression:

Count the total number of balls here. The total number of balls is 12, right?

Now, to find this using multiplication we use:4 x 3 = 3 + 3 + 3 + 3 = 12

That is 3 added to itself 4 times!! This equals 12 which is the amount of balls we counted!

Now, some rules of multiplication:

Multiplication with 0:

Anything multiplied by 0 is 0

Examples:

0 x 0 = 0

0 x 1 = 0

0 x 2 = 0 and so on….

Multiplication with 1:

Anything multiplied by 1 is the number itself

Examples:

0 x 1 = 0

1 x 1 = 1

2 x 1 = 2

3 x 1 = 3

3 x 4 is 4 x 3

First number into second number always equals second number into first number.

Examples:

3 x 4 = 4 x 3 =12

6 x 2 = 2 x 6 = 12

9 x 7 = 7 x 9 = 63

Advanced:

Now, adding 4 x 3 is easy, it can be done with visualizing and splitting it to addition.

But what about bigger numbers?

To do this, we use multiplication of smaller numbers

Let’s learn by using an example:

11 x 21 =?

To solve this, we first write the problem vertically

    11

X 21

-----------------

Now, we first write the multiplication of 11 x 1 which is 11

    11

X 21

-----------------

    11

Now, we go to the next line and write a 0 in the units place.

    11

X 21

-----------------

    11

    0

Now, we multiply 11 x 2 which is 22, and write it before the 0 we just added.

    11

X 21

-----------------

    11

  220

Now, to find 11 x 21, we finally add the two number we just wrote.

    11

X 21

-----------------

      11

+  220

-----------------

231

Therefore, 11 x 21 is equal to 231

This is known as long multiplication.
