गुणाकार

गुणाकार म्हणजे काय?

गुणाकार म्हणजे खूप वेळा बेरीज करणे

हे ‘x’ चिन्हाद्वारे दर्शविले जाते.

गुणाकार करण्यासाठी एखाद्याला बेरीज व्यवस्थित माहित असणे आवश्यक आहे, जे येथे शिकले जाऊ शकते

उदाहरणार्थ

Q] 4 x 3 चे उत्तर काय आहे?

उत्तर: येथे पहिली संख्या 4 आहे आणि दुसरी संख्या 3 आहे

तर, प्रश्न ‘4 क्रॉस 3’ किंवा ‘4 गुणिले 3’ म्हणून वाचला जाऊ शकतो

आता हे सोडवण्यासाठी एक उदाहरण घेऊ:

आपल्याकडे ४ पिशव्यांमध्ये प्रत्येकी 3 चेंडू असल्यास एकूण किती चेंडू आहेत?

येथे 4 पिशव्या आहेत, प्रत्येकाकडे 3 चेंडू आहेत आणि आप्ल्यायला एकूण चेंडू ची संख्या शोधण्याची आवश्यकता आहे

येथे एकूण चेंडूची संख्या मोजा. एकूण चेंडूची संख्या १२ आहे, बरोबर?

आता हे गुणाकार वापरून शोधण्यासाठी: 4 x 3 = 3 + 3 + 3 + 3 = 12

That is 3 added to itself 4 times!! This equals 12 which is the amount of balls we counted!

आता, गुणाकाराचे काही नियमः

0 बरोबर गुणाकार:

0 ने गुणाकार केलेली कोणतीही संख्या 0 आहे

उदाहरणे:

0 x 0 = 0

0 x 1 = 0

0 x 2 = 0

1 बरोबर गुणाकार:

1 ने गुणाकार केलेली कोणतीही संख्या ही संख्या असते

उदाहरणे

0 x 1 = 0

1 x 1 = 1

2 x 1 = 2

3 x 1 = 3

3 x 4 = 4 x 3

पहिली संख्या x दुसरी संख्या =दुसरी संख्या x पहिली संख्या

उदाहरणे:

3 x 4 = 4 x 3 =12

6 x 2 = 2 x 6 = 12

9 x 7 = 7 x 9 = 63

आता प्रगत गुणाकार शिकण्यासाठी आपल्याला काही गुणाकार तक्ते माहित असणे आवश्यक आहे

आपल्याला 1 ते 10 चे गुणाकार तक्त्या माहित असणे आवश्यक आहे.
